Personal Project

My Seoul Dashboard

공공데이터를 활용한 서울 생활 정보 통합 플랫폼

Python Flask | Oracle DB | Leaflet.js

1. 기획 의도

배경 및 해결 문제

  • 날씨, 교통, 지하철 정보가 각기 다른 사이트에 파편화됨
  • 공공데이터 API의 복잡한 응답 구조를 직관적으로 시각화할 필요성

목표 및 효과

  • 지도(Leaflet) 기반의 원스톱 정보 대시보드 구축
  • 데이터 분석(Matplotlib)을 통한 과거 추이 제공
  • 실시간성 보장 및 로컬 캐싱을 통한 성능 최적화

2. 기술 스택

Languages & Frameworks

  • Backend: Python, Flask
  • Frontend: JavaScript, HTML5, CSS3
  • Library: Leaflet.js, Pandas, Matplotlib

Data & DevOps

  • Database: Oracle
  • API: 기상청 예보, 서울시 교통 정보, 지하철 도착 정보
  • Tool: PyCharm

3. 구현 과정

전체 구조

  • Collector: 외부 API 데이터를 정제하여 DB 저장
  • Analyzer: Pandas를 통한 통계 및 그래프 생성
  • API Routes: 클라이언트 요청에 따른 JSON 반환

주요 기능

  • LCC 격자 변환: 위경도를 기상 격자로 좌표 변환
  • Traffic Cache: LocalStorage 기반 TTL 캐싱

4. 결과물

데이터 시각화 (날씨)

Matplotlib을 활용하여 48시간 기온/미세먼지 변화를 이중 막대 그래프로 생성하여 제공

실시간 대시보드 (교통/지하철)

Leaflet 마커 클릭 시 실시간 도착 시간 및 도로 속도를 카드 레이아웃으로 출력

5. 향후 개선 계획 (ML/DL 기반 교통량 예측)

지능형 교통 예측 모델 구축

  • [딥러닝] LSTM 기반 시계열 예측
    도로별 과거 속도 변화 패턴을 학습하여, 30분~1시간 뒤의 미래 정체 상황을 정밀하게 예측.
  • [머신러닝] 날씨-교통 상관관계 분석
    비, 눈, 기온 등 기상 변수가 도로별 통행 속도에 미치는 영향을 데이터로 수치화하여 반영.

데이터 기반 지능형 서비스

  • 선제적 정체 알림
    단순히 "지금 막힌다"가 아니라, "퇴근 정체가 곧 시작되니 지금 출발하세요"라는 예측 가이드를 제공.
  • AI 최적 경로 추천
    날씨에 따른 사고 위험도와 예상 정체 시간을 계산하여, 목적지까지 가장 쾌적하고 빠른 길을 추천.
